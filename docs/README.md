## 快速上手

这是一个使用 VuePress 2 搭建的现代化博客。

### 特性

- 📝 Markdown 支持
- 🎨 Vue 组件
- ⚡ Vite 构建
- 🔍 全文搜索

### 开始使用

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建生产版本
pnpm docs:build
```
